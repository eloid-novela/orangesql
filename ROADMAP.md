# 🚀 OrangeSQL Roadmap

## 🧠 Vision
Build a lightweight, fast, ACID-compliant SQL database engine in modern C++17.

---

# 📌 v1.0 - MVP (Initial Release)

### Core Engine
- [x] Basic SQL parser
- [x] Table creation (CREATE TABLE)
- [x] Insert / Select queries
- [x] Simple in-memory storage

### Storage
- [x] File persistence (basic WAL)
- [ ] Crash recovery system

### Indexing
- [ ] Basic B-Tree implementation

---

# ⚡ v1.1 - Stability & Performance

- [ ] Query optimizer (basic)
- [ ] Better error handling
- [ ] Transaction improvements (ACID full support)
- [ ] Logging system

---

# 🚀 v2.0 - Advanced Engine

- [ ] Full B-Tree indexing engine
- [ ] MVCC (Multi-Version Concurrency Control)
- [ ] Join support (INNER / LEFT)
- [ ] Subqueries
- [ ] Parallel query execution

---

# 🌐 v3.0 - Production Ready

- [ ] Client-server architecture
- [ ] Authentication system
- [ ] Remote connections (TCP API)
- [ ] Docker production deployment
- [ ] Cloud-ready architecture
