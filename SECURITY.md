# 🔐 Security Policy

## Supported Versions

We actively support the latest version of OrangeSQL.

---

## 🚨 Reporting a Vulnerability

If you find a security vulnerability, please DO NOT open a public issue.

Instead, report it privately via:

- GitHub Security tab (recommended)
- Or contact maintainer directly

---

## 📌 What to include

- Description of the issue
- Steps to reproduce
- Possible impact
- Suggested fix (if available)

---

## ⏱ Response Time

We aim to respond within 48–72 hours.

---

## 🛡️ Responsible Disclosure

We appreciate responsible disclosure to help keep OrangeSQL secure.
